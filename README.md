# 901 ULTRA League Homepage

Static one-page site for the 901 ULTRA League with a Discord call-to-action and space for a QR code.

## Usage
- Invite link is set to `https://discord.gg/kbVHK566sM`; update it in `index.html` if you rotate the invite.
- Drop your QR image as `discord-qr.png` in the repo root to have it render in the CTA card (placeholder hides automatically).
- Deploy with GitHub Pages by serving `index.html` from the default branch.
